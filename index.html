Fonctionnalites

Ce que le code fait reellement

Pas de marketing. Fonctionnalites verifiables dans le depot.

Detection de categorie

Mots-cles avec frontieres re.boundary : food, tech, service, ecom, immobilier.

Sanitisation des inputs

Echappement HTML, limite 500 caracteres, validation numeros algeriens 05xx/06xx/07xx.

Logo SVG local

Generation SVG avec gradient et initiales. Zero appel reseau, zero service etranger.

Serveur leads asyncio

HTTP sur 127.0.0.1:8080. POST /save-lead, GET /leads. Capture reelle des prospects.

Ollama fallback

Generation via Ollama local (localhost:11434). Si indisponible, template souverain automatique.

Agent gestion erreurs

Capture, classe, statistiques et persistence JSON. safe_run() dans la forge pour ne jamais crasher.

Formulaire CIB/Edahabia
